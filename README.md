# FTP Client for CS3700

## Overview
This FTP Client is a Python-based command-line tool that facilitates interaction with FTP servers. It supports operations such as connecting to an FTP server, listing directory contents, creating and removing directories, copying files between the server and the local file system, and setting transfer modes. This tool is designed for developers and system administrators who need to automate FTP operations or integrate FTP functionality into their scripts or applications.

## Features
- Connect to FTP servers using credentials or anonymously.
- Parse FTP URLs and responses for easy interaction.
- Perform directory operations: create (`mkdir`), remove (`rmdir`), and list contents (`ls`).
- File operations: copy (`cp`), remove (`rm`).
- Supports setting binary and stream transfer modes for file transfers.
